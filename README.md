## 🚀 Overview
Welcome to my Symfony playground! This repository is dedicated to exploring, testing, and documenting the core features and advanced mechanics of the Symfony framework. It serves as a living laboratory for my learning journey.

Instead of just following the standard way, I dive into the underlying components to understand how they work under the hood.

## 📚 Featured Learning Modules

### 🛡️ Security: 

#### [Manual CSRF Protection](https://github.com/dominicchinkh/tutorial_symfony/wiki/Manual-CSRF-Protection-in-Symfony)
Understanding how Symfony defends against Cross-Site Request Forgery (CSRF). This section explores the manual implementation of tokens for scenarios where the standard Form Component isn't used.

### 🔌 API & Routing: 

#### [Leveraging HTTP Verbs](https://github.com/dominicchinkh/tutorial_symfony/wiki/Leveraging-HTTP-Verbs-in-Symfony-Forms)
HTML forms are traditionally limited to GET and POST. In this module, I experiment with how Symfony bypasses these limitations to support modern RESTful architectures.

## 🛠️ Setup & Usage
Clone the repository:

```Bash
git clone https://github.com/dominicchinkh/tutorial_symfony.git
```

Install dependencies:

```Bash
composer install
```

Start the local server:

```Bash
symfony serve
```

## 📖 About Symfony
[Symfony](https://symfony.com/) is a set of reusable PHP components and a PHP framework for web projects. It is built on a philosophy of decoupling and standardizing code, making it one of the most robust foundations for enterprise-level PHP applications.
